Расширение функционала библиотеки jquery (а так же любых библиотек на её основе) "на лету" без изменений исходного кода осуществляется с помощью свойства ".fn", которое является алиасом свойства js-объектов "prototype" и служит для реализации методики наследования методов класса-прототипа. В jquery есть собственная "родная" функции для модификации методов - .extend().
